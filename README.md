# dqnet
Dual quaternions library for .NET

This library is a port of the libdq C library to C#.
It also re-used some code taken from "A Beginners Guide to Dual-Quaternions", and is inspired by the SharpDX Quarternion class.

Links:

The C libdq library - http://www.iri.upc.edu/people/esimo/code/libdq/
A Beginners Guide to Dual-Quaternions - http://wscg.zcu.cz/wscg2012/short/a29-full.pdf
SharpDX - http://sharpdx.org/
